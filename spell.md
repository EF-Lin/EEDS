# 咒語彙整
整理各種幫助 AI 閱讀pdf、整理數據的咒語（spell）。

## For ChatGPT
```md
請根據我提供的論文 PDF，盡可能為 12550.csv 中每一個樣品補完空白欄位，並以 SAMPLE NAME 為唯一對照鍵（只用同名樣品對應，不可用推測或其他線索硬配）。

【核心規則】（必須遵守）
- 如果原先欄位已有數值/文字/格式，完全不要更動（包含內容與字型/格式）。不可新增欄位，只允許在「原本是空白」的欄位填入資料。
- 你只能填寫論文中明確出現的資訊（含正文、表格、圖說、附錄等）。禁止推測、延伸、換算推估、或用常識補齊。
- 若某欄位資訊在論文中未出現，該欄位留空。
- 若 CSV 既有數據與論文內容不一致，不要改，但請在最後彙整列出差異：包含 SAMPLE NAME、欄位名稱、CSV 原值、論文對應值、論文出處位置。

【優先需要主動找並補的資訊類型】
(1) 定年與時間
- 年齡值、誤差、定年方法
- 年齡上下限（AGE MIN / MAX）
- 地質時代與細分
- 歷史噴發年份／月份（若有）

(2) 位置與採樣條件
- 地名與構造背景
- 緯度／經度（含 MIN / MAX）
- 海拔高度（含 MIN / MAX）
- 採樣方式與陸／海環境

(3) 樣品與岩性描述
- 樣品名稱與註解
- 岩石大類、岩名、組織
- 材料型態
- 變質或蝕變狀態

(4) 化學組成（地化資料）
- 主量元素（所有氧化物與揮發分）
- 微量元素（含稀土元素）

(5) 同位素與放射性系統
- Sr–Nd–Pb–Hf–Os 同位素比值
- Re–Os、Rb–Sr、Sm–Nd、Lu–Hf 比值
- 惰性氣體同位素（He–Ne–Ar）
- 放射性元素/同位素濃度（Ra、Pa、U、Th 等）

【特別要求：盡量補的 CSV 欄位範圍】
- C 欄：SAMPLE NAME（若空白且論文有明確名稱才填）
- G–L 欄：LATITUDE (MIN.), LONGITUDE (MIN.), LATITUDE (MAX.), LONGITUDE (MAX.), Revise_LAT(THIS STUDY), Revise_LON(THIS STUDY)
- Q 欄：ROCK TYPE（只能填 VOLCANIC ROCK 或 PLUTONIC ROCK）
- U–X 欄：AGE (MIN.), AGE (MAX.), AGE (MA), AGE ERROR (2 sigma)
- Y 欄：DATING METHOD
- Z 欄：TECTONIC SETTINGS
- AG–DQ 欄：所有地化資料欄位（主量/微量/稀土等）

【交付內容】（最後附加給我）
- 你實際填入了哪些資料（逐筆或至少逐樣品逐欄位列出）。
- 每一筆填入資料的論文來源位置：至少包含「頁碼 + 表格/圖號/段落（例如 Table 3、Section 4.2、Fig. X caption）」；必須可回溯驗證。
- CSV 與論文不一致清單（依上面規則，不可直接改 CSV 原值）。
- 其他你注意到的異常情況。

最後直接將數據結合到原先的csv中讓我下載。
```
