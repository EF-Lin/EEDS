# 咒語彙整
整理各種幫助 AI 閱讀pdf、整理數據的咒語（spell）。

## For ChatGPT
```md
請根據我提供的論文pdf盡可能為csv檔中的「每一個樣品」補完列表欄位，以下為規則和指引。

【規則】
- 如果原先欄位已有數值則跳過即可。
- 你只能填寫論文中「明確出現」的資訊，不可加入推測內容。
- 若某一格資訊在論文中未出現，則該欄位留空。

【指引】
你需要特別主動尋找的資訊類型有：
1) 定年與時間
   - 年齡值、誤差、定年方法
   - 年齡上下限（min / max）
   - 地質時代與細分
   - 噴發年份／月份（若為歷史噴發）

2) 位置與採樣條件
   - 地名與構造背景
   - 緯度／經度（含 min / max）
   - 海拔高度（含 min / max）
   - 採樣方式與陸／海環境

3) 樣品與岩性描述
   - 樣品名稱與註解
   - 岩石大類、岩名、組織
   - 樣品材料型態
   - 變質或蝕變狀態

4) 化學組成
   - 主量元素（所有氧化物與揮發分）
   - 微量元素（含稀土元素）

5) 同位素與放射性系統
   - Sr–Nd–Pb–Hf–Os 同位素比值
   - Re–Os、Rb–Sr、Sm–Nd、Lu–Hf 親子比
   - 惰性氣體同位素（He–Ne–Ar）
   - 放射性元素與同位素濃度（Ra、Pa、U、Th 等）

最後請告訴我你填充的資料有哪些、分別來自論文何處。
```
